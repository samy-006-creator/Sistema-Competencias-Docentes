 Sistema de Gestión de Competencias Docentes

Descripción

Este proyecto consiste en el desarrollo de un Sistema de Gestión de Competencias Docentes utilizando Go (Golang). El sistema permite administrar información relacionada con docentes y sus competencias académicas mediante la aplicación de principios de programación funcional.

Objetivo

Desarrollar un sistema que permita gestionar docentes y competencias académicas utilizando programación funcional en Go.

Funcionalidades

* Registro de docentes.
* Gestión de competencias académicas.
* Filtrado de docentes por nivel de competencia.
* Actualización de niveles de competencia.
* Generación de estadísticas académicas.
* Aplicación de programación funcional mediante Filter, Map y Reduce.

Tecnologías Utilizadas

* Go (Golang)
* Git
* GitHub

Programación Funcional Implementada

### Filter

Permite filtrar docentes según criterios específicos, como el nivel de competencia.

Map

Permite transformar los datos de los docentes generando nuevos registros sin modificar los originales.

Reduce

Permite acumular información para obtener estadísticas y resultados globales.
